## 01 — Строительный кодекс (Planning & Building Code)

### 1.1 Decree No. (45) of 2021 — Dubai Building Code
| | |
|---|---|
| **Что это** | Декрет Правителя Дубая, который вводит единый Dubai Building Code (DBC) как обязательный стандарт |
| **Зачем нужен** | Правовая основа: без этого декрета DBC — просто рекомендация. Декрет придаёт ему силу закона |
| **Ключевое** | Применяется ко ВСЕМ проектам в Дубае, включая free zones и DIFC. Отменяет все противоречащие нормы |
| **Объём** | 4 страницы, 9 статей |
| **Статус** | Действующий (с 18 декабря 2021) |
| **PDF в папке** | [01_Planning_and_Building_Code/Decree_45_2021_Dubai_Building_Code.pdf](01_Planning_and_Building_Code/Decree_45_2021_Dubai_Building_Code.pdf) |
| **Официальный источник** | [Dubai Legal Portal (dlp.dubai.gov.ae)](https://dlp.dubai.gov.ae/Legislation%20Reference/2021/Decree%20No.%20(45)%20of%202021%20Concerning%20the%20Dubai%20Building%20Code.pdf) |

### 1.2 Dubai Building Code 2021 (полный технический кодекс)
| | |
|---|---|
| **Что это** | Сам строительный кодекс Дубая: архитектура, конструкции, пожарная безопасность, энергоэффективность, доступность, MEP |
| **Зачем нужен** | Технические требования к проектированию — без соответствия DBC не получить разрешение на строительство |
| **Ключевое** | Единый кодекс, заменивший разрозненные нормы. Введён Decree No. 45/2021 |
| **Статус** | Действующий |
| **PDF в папке** | [01_Planning_and_Building_Code/Dubai_Building_Code_2021_DDA.pdf](01_Planning_and_Building_Code/Dubai_Building_Code_2021_DDA.pdf) |

## 04 — Зонирование и застройка (Zoning & Land Use)

### 4.1 Local Order No. (2) of 1999 — Classification and Regulation of Land Use
| | |
|---|---|
| **Что это** | Базовый закон о зонировании Дубая: классификация территорий, виды разрешённого использования, система Planning Permit |
| **Зачем нужен** | Определяет что можно строить на конкретном участке. Без соответствия зонированию — нельзя получить разрешение |
| **Ключевое** | Permitted Uses / Conditional Uses / Non-compliant Uses. Карты зон: R1–R9 (жильё), C1–C2 (коммерция) и др. |
| **Объём** | 95 страниц |
| **Статус** | Действующий (базовый акт, дополнялся) |
| **PDF в папке** | [04_Zoning_and_Land_Use/Local_Order_2_1999_Land_Use.pdf](04_Zoning_and_Land_Use/Local_Order_2_1999_Land_Use.pdf) |
| **Официальный источник** | [Dubai Legal Portal (dlp.dubai.gov.ae)](https://dlp.dubai.gov.ae/Legislation%20Reference/1999/Local%20Order%20No.%20(2)%20of%201999.pdf) |

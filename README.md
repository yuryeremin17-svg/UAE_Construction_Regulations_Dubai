# Нормативные документы строительства — Дубай / ОАЭ

Пакет ключевых регуляторных документов по строительству в эмирате Дубай.

## Как пользоваться

**Откройте каталог документов:**

### [Открыть каталог документов](https://yuryeremin17-svg.github.io/UAE_Construction_Regulations_Dubai/00_CONTENTS.html)

Вы увидите описание каждого документа и синие кнопки **«Скачать PDF»** — нажимайте, файлы скачиваются сразу.
